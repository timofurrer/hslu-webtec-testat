Testat Aufgabe

Diese Webseite ist Teil der Testat Aufgabe fuer das HSLU WebTec Modul.

Zum Source

Aufgabe

Die Aufgabenstellung fuer das HSLU WebTec Testat:

Es ist eine WebSite zu erstellen, die den untenstehenden Ansprüchen genügen muss. Der Inhalt dieser WebSite ist egal. Diese WebSite ist schlussendlich auch online verfügbar zu machen damit ich sie kontrollieren kann.

Aus dem original Auftrag

Home

Auf der Startseite sind allgemeine Informationen ueber diese Testat Webseite zu finden.

Canvas

Die Canvas-Seite beinhaltet eine kleine Demonstration, wie man ein Canvas-Element vom Server laden und auf der Webseite per Canvas Rendern kann.

Formular

Auf der Formular-Seite befindet sich ein Kontaktformular mit Clientseitiger Eingabepruefung. Die Metainformationen fuer das Formular werden vom Server geladen.

Texte und Bilder

Die Texte und Bilder Seite enthaelt vom Server geladene Texte und Bilder.
